## 📦 DeleteConfirmModal 组件说明

### 组件位置
`components/Shared/DeleteConfirmModal.tsx`

### Props 接口

```typescript
interface DeleteConfirmModalProps {
  isOpen: boolean;
  onClose: () => void;
  onConfirm: () => void;
  title?: string;
  description?: string;
  warningText?: string;
  itemName?: string;
  confirmText?: string;
  cancelText?: string;
  isLoading?: boolean;
  size?: 'xs' | 'sm' | 'md' | 'lg' | 'xl' | '2xl' | '3xl' | '4xl' | '5xl' | 'full';
  showIcon?: boolean;
}
```

### 默认值

- `size`: 'sm'
- `showIcon`: true
- `isLoading`: false

### 使用示例

```typescript
import { DeleteConfirmModal } from '../components/Shared/DeleteConfirmModal';

// 基础用法
<DeleteConfirmModal
  isOpen={isDeleteOpen}
  onClose={onDeleteClose}
  onConfirm={confirmDelete}
/>

// 带自定义文本
<DeleteConfirmModal
  isOpen={isDeleteOpen}
  onClose={onDeleteClose}
  onConfirm={confirmDelete}
  title="删除脚本？"
  description="确定要删除这个脚本及其相关内容吗？"
  confirmText="确认删除"
  cancelText="再想想"
/>

// 不显示图标（分镜管理页风格）
<DeleteConfirmModal
  isOpen={isDeleteOpen}
  onClose={onDeleteClose}
  onConfirm={confirmDelete}
  showIcon={false}
  size="md"
/>
```

#### 文件位置

#### 当前问题代码（656-707行）
```typescript

  isOpen={isDeleteOpen}
  onOpenChange={onDeleteOpenChange}

  <ModalContent>
    {onClose => (
      <>
        <ModalHeader className="flex flex-col gap-1 items-center pt-8">
          <div className="w-16 h-16 bg-red-50 dark:bg-red-900/20 text-red-500 rounded-full flex items-center justify-center mb-2">
            <AlertTriangle className="w-8 h-8" />

            {t.project.deleteConfirmTitle}

            {t.project.confirmDelete}
          </p>

            variant="light"
            onPress={onClose}
            className="flex-1 font-bold text-slate-500 h-12 rounded-2xl"

          </Button>
          <Button
            color="danger"
            variant="shadow"

```

**第二步：替换弹窗代码**
将656-707行的旧弹窗代码替换为：
```typescript

  isOpen={isDeleteOpen}
  onClose={onDeleteClose}
  onConfirm={confirmDelete}
  title={t.project.deleteConfirmTitle}
  description={t.project.confirmDelete}
  confirmText={t.project.deleteConfirmAction}
  cancelText={t.dashboard.cancel}

```

**注意**：需要确认 `onDeleteClose` 是否可用，如果使用的是 `onDeleteOpenChange`，需要调整为：
```typescript

  isOpen={isDeleteOpen}
  onClose={() => onDeleteOpenChange(false)}

  title={t.project.deleteConfirmTitle}
  description={t.project.confirmDelete}
  confirmText={t.project.deleteConfirmAction}
  cancelText={t.dashboard.cancel}
/>
```
