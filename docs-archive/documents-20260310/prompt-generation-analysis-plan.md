# 提示词生成逻辑深度分析计划

## 问题描述

从截图可以看到：

**角色面板显示的提示词**：

```
年轻女性约22岁，清秀面容，普通长发自然垂肩，身着简洁的职业实习生装扮，白色衬衫搭配深色西裤，体型苗条中等身高，手中常抱着文件或笔记本，整体形象干净利落。
```

**场景面板显示的提示词**：

```
现代商务办公室内景，江哲坐在办公桌后，苏晴站在桌前整理散落的文件。办公室采用冷色调装修，桌上摆放着笔记本电脑和文件堆，光线从天花板灯具洒下，营造紧张的工作氛围。
```

用户反馈：这些提示词"不符合用户需求"

## 分析目标

1. **理解提示词来源**：这些提示词是从哪里生成的？
2. **理解解析流程**：大模型是如何解析提取这些描述的？
3. **找出问题所在**：为什么生成的提示词不符合用户需求？
4. **提出改进方案**：如何优化提示词生成逻辑？

## 当前发现的线索

### 1. 角色提示词来源

**文件**: `services/scriptParser.ts` 第138行

```typescript
"visualPrompt": "仅描述外貌特征：性别、年龄、面容、发型、服饰、体型、标志性物品。禁止：剧情、情绪、背景、关系。格式示例：古代少女约16岁，瓜子脸柳叶眉丹凤眼，乌黑长发挽成垂鬟分肖髻，身着淡紫色绣花襦裙外披白色纱衣，体态纤细，手持团扇。80-120字"
```

LLM被指示生成 `visualPrompt` 字段，这个字段会被用作角色的 `prompt`。

### 2. 场景提示词来源

**文件**: `services/scriptParser.ts` 第211行

```typescript
"visualPrompt": "用于AI生图的描述，50字以内"
```

LLM被指示生成 `visualPrompt` 字段用于场景生图。

### 3. 资产创建流程

需要追踪：

- `visualPrompt` 如何从解析结果传递到资产对象
- 资产创建时 `prompt` 字段如何被填充

## 实施步骤

### 阶段1: 追踪角色提示词数据流

1. **查找角色资产创建代码**
   - 搜索 `CharacterAsset` 创建逻辑
   - 查找 `visualPrompt` 如何映射到 `prompt`

2. **分析提示词传递路径**

   ```
   剧本解析 → LLM生成visualPrompt → ScriptCharacter.visualPrompt
   → 创建CharacterAsset → asset.prompt
   → CharacterDetail显示
   ```

3. **定位问题**
   - 检查 `visualPrompt` 是否被正确传递
   - 检查是否有其他逻辑覆盖/修改了prompt

### 阶段2: 追踪场景提示词数据流

1. **查找场景资产创建代码**
   - 搜索 `SceneAsset` 创建逻辑
   - 查找 `visualPrompt` 如何映射到 `prompt`

2. **分析提示词传递路径**

   ```
   剧本解析 → LLM生成visualPrompt → ScriptScene.visualPrompt
   → 创建SceneAsset → asset.prompt
   → SceneDetail显示
   ```

3. **定位问题**
   - 场景提示词包含剧情描述（"江哲坐在办公桌后..."）
   - 这可能不符合"场景设定图"的需求

### 阶段3: 分析问题根源

**可能的问题**：

1. **角色提示词问题**
   - `visualPrompt` 可能没有被正确传递到 `asset.prompt`
   - 或者使用了错误的字段（如 `description` 而非 `visualPrompt`）

2. **场景提示词问题**
   - LLM生成的 `visualPrompt` 包含剧情动作（"江哲坐在..."）
   - 但场景设定图应该只是环境描述，不包含人物动作
   - 提示词模板可能需要优化

### 阶段4: 提出改进方案

**方案选项**：

**选项A: 修复数据传递**

- 确保 `visualPrompt` 正确传递到 `asset.prompt`

**选项B: 优化LLM提示词**

- 修改 `PROMPTS.character` 和 `PROMPTS.scene`
- 更明确地指导LLM生成符合需求的描述

**选项C: 允许用户编辑**

- 在UI中明确标识这是"AI生成的提示词"
- 提供编辑功能让用户修改

**选项D: 组合方案**

- 修复数据传递 + 优化LLM提示词 + 允许用户编辑
