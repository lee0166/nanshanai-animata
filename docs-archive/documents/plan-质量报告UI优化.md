### 步骤3: 实现质量维度评分可视化

**目标**: 添加进度条展示各维度得分

**实现内容**:

1. 导入 HeroUI Progress 组件
2. 创建维度评分展示组件
3. 实现得分动画（从0到实际得分的过渡）
4. 根据得分显示不同颜色（优秀/良好/需改进）

**代码示例**:

```tsx
// 维度评分展示
<div className="space-y-3">
  {dimensions.map(dim => (
    <div key={dim.name} className="flex items-center gap-3">
      <span className="w-20 text-sm">{dim.label}</span>
      <Progress value={dim.score} color={getScoreColor(dim.score)} className="flex-1" size="sm" />
      <span className="w-12 text-right font-medium">{dim.score}分</span>
    </div>
  ))}
</div>
```

**预计工期**: 40分钟
