# 情绪曲线图表优化计划

## 当前问题分析

1. **颜色单调** - 整个图表统一使用黄色 (warning)，缺乏变化
2. **样式简陋** - 填充区域半透明黄色，曲线也是单色
3. **视觉层次不分明** - 数据点和情节标签不够突出
4. **情节点卡片** - 样式可以更精致

## 优化方案

### 1. 颜色优化

- 使用渐变色填充（从高情绪到低情绪）
- 曲线颜色更加柔和（改用蓝紫色调）
- 数据点根据情绪类型着色

### 2. 样式优化

- 添加网格背景
- 美化曲线（使用更柔和的描边）
- 数据点添加光晕效果

### 3. 布局优化

- 情节点卡片使用更精致的样式
- 优化间距和排列

## 预期效果

```
优化前：单一黄色曲线
优化后：渐变蓝紫色曲线 + 彩色情节点 + 精致卡片
```
