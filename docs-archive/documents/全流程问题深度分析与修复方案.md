## 修复方案

### 方案：加载并显示参考图缩略图

**实施步骤：**

#### 1. 添加状态管理

在 ShotManager.tsx 中添加：

```typescript
// 参考图URL缓存
const [referenceImageUrls, setReferenceImageUrls] = useState<{
  character?: string;
  scene?: string;
}>({});
```

#### 2. 加载参考图URL

当选择关键帧时，加载角色和场景的图片URL：

```typescript
useEffect(() => {
  if (selectedKeyframeIndex >= 0 && selectedShot?.keyframes[selectedKeyframeIndex]?.references) {
    loadReferenceImageUrls();
  }
}, [selectedKeyframeIndex, selectedShot]);

const loadReferenceImageUrls = async () => {
  const refs = selectedShot.keyframes[selectedKeyframeIndex].references;
  const urls: { character?: string; scene?: string } = {};

  // 加载角色图URL
  if (refs?.character?.id) {
    const charAsset = allCharacterAssets.find(c => c.id === refs.character!.id);
    if (charAsset?.filePath) {
      urls.character = await storageService.getAssetUrl(charAsset.filePath);
    }
  }

  // 加载场景图URL
  if (refs?.scene?.id) {
    const sceneAsset = allSceneAssets.find(s => s.id === refs.scene!.id);
    if (sceneAsset?.filePath) {
      urls.scene = await storageService.getAssetUrl(sceneAsset.filePath);
    }
  }

  setReferenceImageUrls(urls);
};
```
