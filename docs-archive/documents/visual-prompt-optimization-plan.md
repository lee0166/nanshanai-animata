## 问题分析

### 核心问题

文本模型生成的视觉提示词与AI生图实际需求不匹配，导致：

1. 提示词包含非必要元素，增加用户修改成本
2. 场景提示词定位偏差，偏向分镜描述而非场景素材描述
3. 物品提示词存在类似问题

### 具体问题分类

#### 1. 角色视觉提示词问题

**示例**：

- 角色：林晚（26岁女性，清秀憔悴）
- 当前生成："年轻女性约26岁，面容清秀，略显憔悴...手中掌握着一串家门钥匙"
- **问题**："家门钥匙"是非核心道具，不贯穿故事，不彰显角色特质

**理想提示词**：
"年轻女性，约26岁，面容清秀，略显憔悴。眉眼间透着疲惫，黑色长发随意挽起，身着简约深色通勤外套搭配休闲长裤，体态纤细单薄，神情安静"

#### 2. 场景视觉提示词问题

**示例**：

- 场景：深秋傍晚的居民楼道
- 当前生成："深秋傍晚，昏暗的居民楼道，一个就是乡里传说着一只。小奶猫，琥珀色的眼睛。怯生生的亮着地面，散落几片枯叶，氛围清冷。"
- **问题**：包含分镜细节（小奶猫、眼睛），而非纯场景描述

**理想提示词**：
"深秋傍晚，昏暗的居民楼道，墙面斑驳，地面散落几片枯叶，光线昏黄，氛围清冷，写实风格"

#### 3. 物品视觉提示词问题

- 存在与角色、场景类似的问题
- 需要具体分析

## 优化方案

### 方案一：优化 Prompt 工程（推荐）

#### 1.1 角色视觉提示词优化

**优化后的 Prompt 模板**：

```
为角色生成AI生图视觉提示词，要求：
1. 只包含角色的核心视觉特征：
   - 年龄、性别、面容特征
   - 发型、发色
   - 标志性服装风格（非临时穿着）
   - 体态、气质
   - 标志性配饰（如有）
2. 排除以下内容：
   - 临时持有的道具
   - 非标志性的动作姿态
   - 场景相关的临时状态
   - 情绪化的临时表情（除非是核心特征）
3. 提示词应简洁、具象，便于AI理解
4. 使用英文逗号分隔的关键词形式

角色信息：{角色详细描述}
```

#### 1.2 场景视觉提示词优化

**优化后的 Prompt 模板**：

```
为场景生成AI生图视觉提示词，要求：
1. 只描述场景本身的环境特征：
   - 时间（清晨、正午、黄昏、夜晚）
   - 地点类型（室内/室外，具体环境）
   - 光线条件（自然光、人工光、昏暗、明亮）
   - 环境细节（建筑、植被、天气、氛围）
   - 色调、风格
2. 排除以下内容：
   - 任何角色或生物
   - 动态动作或事件
   - 分镜式的画面描述
   - 故事情节相关的临时元素
3. 提示词应描述一个静态的、可作为背景使用的场景
4. 使用英文逗号分隔的关键词形式

场景信息：{场景详细描述}
```

#### 1.3 物品视觉提示词优化

**优化后的 Prompt 模板**：

```
为物品生成AI生图视觉提示词，要求：
1. 只描述物品本身的特征：
   - 物品类型、材质
   - 颜色、纹理
   - 形状、大小
   - 风格、年代感
   - 标志性细节
2. 排除以下内容：
   - 使用场景（除非必要）
   - 与角色的互动
   - 动态效果
   - 故事相关的临时状态
3. 提示词应聚焦于物品本体，便于单独生成
4. 使用英文逗号分隔的关键词形式

物品信息：{物品详细描述}
```

## 预期效果

1. **角色提示词**：
   - 只包含核心视觉特征
   - 排除临时道具和动作
   - 用户无需频繁修改

2. **场景提示词**：
   - 聚焦场景环境本身
   - 排除角色和动态元素

   - 聚焦物品本体特征
   - 排除使用场景描述
   - 便于单独生成
