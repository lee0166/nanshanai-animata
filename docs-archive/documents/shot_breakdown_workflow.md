# 分镜管理 - 关键帧拆分工作流程研究计划

## 目标

深度了解分镜管理中每个分镜头的拆分关键帧工作流程，以及流程中每一个环节所使用到的系统提示词和用户提示词。

## 研究范围

### 1. 分镜数据结构分析

- 分镜(Shot)的数据模型定义
- 分镜与场景(Scene)、角色(Character)的关系
- 分镜的元数据信息（景别、镜头运动、时长等）

### 2. 关键帧拆分流程

- 分镜如何拆分为关键帧
- 关键帧的生成逻辑
- 关键帧与分镜的对应关系

### 3. 提示词系统

- 系统提示词（System Prompt）设计
- 用户提示词（User Prompt）构建
- 提示词与分镜属性的映射关系

### 4. 生成流程

- 从分镜描述到关键帧的完整流程
- 涉及的AI模型调用
- 生成结果的验证和存储
