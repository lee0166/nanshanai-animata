## 一、问题背景与诊断

### 1.1 日志分析发现的问题

**测试日志**：`localhost-1775984683322.log`

**关键错误**：
```

[ScriptParser] Final result: 26 shots (25 key + 1 optional) in 8 batches
目标：80 shots (56 key + 24 optional)
完成率：32.5%（26/80）
```

### 1.2 问题影响链

```
场景内容提取失败（0 字符）
  ↓
ScriptScene.visualPrompt 使用默认值："林薇的办公室的场景画面"（13 字）
  ↓
用户进入场景管理 → 场景详情页 → 生图工作流
  ↓
调用 getSceneImagePrompt("林薇的办公室的场景画面", "办公室")
  ↓
AI 生图提示词质量极差
  ↓
无法生成影视级场景图
  ↓
用户需要手动补充描述
```

## 二、项目真实流程梳理

### 2.1 完整的资产生图流程

```
阶段 1：小说解析 (scriptParser.ts)
  ↓
  生成 ScriptCharacter[]、ScriptScene[]、ScriptItem[]
  ↓
  每个资产包含 visualPrompt 字段
  ↓
  ❌ 问题：visualPrompt 质量为默认值（7-13 字）
  
阶段 2：资产管理（用户操作）
  ↓
  用户点击"角色管理"Tab
  ↓
  查看角色列表，选择某个角色
  ↓
  进入角色详情页（资产生图工作流）
  
阶段 3：资产生图工作流（三阶段）
  ↓
  Phase 1: 生成面部特征图
    - 使用 getFacePortraitPrompt(userPrompt)
    - userPrompt = ScriptCharacter.visualPrompt
    - ❌ 问题：visualPrompt 只有"林薇的角色形象"（7 字）
    
  ↓ Phase 2: 生成全身正面视图
    - 使用 getFullBodyPrompt(userPrompt)
    - userPrompt = ScriptCharacter.visualPrompt
    - ❌ 问题：同样的低质量提示词
    
  ↓ Phase 3: 生成多角度视图
    - 以全身图为参考
    - 生成侧面、背面、四分之三侧面
    
阶段 4：场景和物品类似流程
  ↓
  场景：生成全景图→广角图→细节图
  物品：生成主视图→细节图→多视角图
  
阶段 5：分镜生成
  ↓
  基于已生成的角色/场景/物品资产
  ↓
  生成分镜图（包含人物、场景）
```

## 三、核心问题定位

### 3.1 visualPrompt 的生成逻辑

**当前代码**（scriptParser.ts 第 2870 行和 2894 行）：

```typescript
// 场景的 visualPrompt 生成
private validateScene(scene: Partial<ScriptScene>, name: string): ScriptScene {
  return {
    // ...
    visualPrompt: scene.visualPrompt || `${name}的场景画面`,  // ❌ 简单默认值
  };
}

// 角色的 visualPrompt 生成
private validateCharacter(character: Partial<ScriptCharacter>, name: string): ScriptCharacter {
  return {
    // ...
    visualPrompt: character.visualPrompt || `${name}的角色形象`,  // ❌ 简单默认值
  };
}
```

**问题**：
1. ❌ 如果 LLM 返回的 `visualPrompt` 为空，使用简单默认值
2. ❌ 默认值只有 7-13 个字，无法生成高质量生图
3. ❌ 场景内容提取失败时，`visualPrompt` 质量极差

### 3.2 Prompt 设计的问题

**当前 `PROMPTS.scenesBatch`**（第 1052-1114 行）：

```typescript
{
  "name": "场景名",
  "description": "场景环境描述，不包含人物动作",
  "visualPrompt": "【系统字段，将由程序自动生成，无需填写】"  // ❌ 问题：告诉 LLM 无需填写
}
```

**问题**：
1. ❌ Prompt 告诉 LLM"无需填写"visualPrompt
2. ❌ 导致 LLM 返回空值
3. ❌ 代码使用默认值填充
4. ❌ 生图质量无法保证

### 4.3 P1 修复：增强 validate 逻辑

#### **修改 `validateScene` 方法**

**位置**：`services/scriptParser.ts` 第 2870 行附近

**修改前**：
```typescript
private validateScene(scene: Partial<ScriptScene>, name: string): ScriptScene {
  return {
    name: scene.name || name,
```

**修改后**：
```typescript
private validateScene(scene: Partial<ScriptScene>, name: string): ScriptScene {
  const validated = {
    name: scene.name || name,
    description: scene.description || name,
    // ✅ 基于 environment、timeOfDay、weather 等生成丰富的 visualPrompt
    visualPrompt: scene.visualPrompt || this.generateSceneVisualPrompt(scene),
  };
  
  return validated;
}

/**
 * 基于场景信息生成 visualPrompt
 */
private generateSceneVisualPrompt(scene: Partial<ScriptScene>): string {
  const parts: string[] = [];
  
  // 收集所有可用的视觉元素
  if (scene.environment?.architecture) {
    parts.push(scene.environment.architecture);
  }
  if (scene.environment?.furnishings && scene.environment.furnishings.length > 0) {
    parts.push(`陈设：${scene.environment.furnishings.join(', ')}`);
  }
  if (scene.environment?.lighting) {
    parts.push(`光线：${scene.environment.lighting}`);
  }
  if (scene.environment?.colorTone) {
    parts.push(`色调：${scene.environment.colorTone}`);
  }
  if (scene.timeOfDay) {
    parts.push(`时间：${scene.timeOfDay}`);
  }
  if (scene.season) {
    parts.push(`季节：${scene.season}`);
  }
  if (scene.weather) {
    parts.push(`天气：${scene.weather}`);
  }
  
  // 如果没有可用元素，返回基础描述
  if (parts.length === 0) {
    return `${scene.name}的场景，详细环境描述待补充`;
  }
  
  // 组合成完整的 visualPrompt
  return parts.join(', ');
}
```

**优势**：
1. ✅ 基于已有信息生成丰富的 visualPrompt
2. ✅ 即使 LLM 未返回 visualPrompt，也能生成可用的描述
3. ✅ 不再依赖硬编码的模板

### 4.4 P2 修复（可选）：使用 LLM 优化 visualPrompt

```typescript
/**
 * 使用 LLM 生成高质量 visualPrompt
 */
async generateHighQualityVisualPrompt(
  type: 'scene' | 'character',
  data: ScriptScene | ScriptCharacter
): Promise<string> {
  const prompt = `
请根据以下${type === 'scene' ? '场景' : '角色'}信息，生成用于 AI 生图的视觉提示词：

【${type === 'scene' ? '场景' : '角色'}名称】
${data.name}

【${type === 'scene' ? '环境' : '外貌'}信息】
${JSON.stringify(data, null, 2)}

请生成一段 100 字以内的视觉描述，用于 AI 生图。
要求：
1. 包含所有关键视觉元素
2. 描述具体、生动
3. 适合 AI 理解
4. 达到影视级生图标准
`;

  const response = await this.callLLM(prompt, 'visual');
  return response;
}
```
