# 智能记忆状态展示方案（参考业界实践）

## 你的核心需求

1. **解析完成后**：在结果页面显示智能记忆是否开启
2. **重新解析时**：让用户知道当前配置，并可调整

## 业界常见做法参考

### 做法 1：状态标签 + 工具提示（推荐）

**参考**：VS Code 扩展状态栏、GitHub Actions 状态标识

**实现**：

- 在解析结果卡片右上角添加状态标签
- 鼠标悬停显示详细配置信息

```
┌─────────────────────────────────────┐
│ 解析结果                    [🧠 智能记忆] │  ← 标签显示
│                                     │
│ 小说：混沌仙途                      │
│ 字数：16,189                        │
│ 角色：3个  场景：4个  分镜：31个     │
│                                     │
│ [重新解析] [导出结果]               │
└─────────────────────────────────────┘

悬停提示：
- 智能记忆：已启用
- ChromaDB：已连接
- 向量维度：384
- 集合：script_memory
```

**优点**：

- 信息密度高，不占用过多空间
- 用户随时可见当前状态
- 符合现代 UI 设计趋势

### 做法 2：配置摘要栏（参考 Figma、Notion）

**参考**：Figma 导出设置摘要、Notion 页面属性

**实现**：

- 在结果页面上方添加固定摘要栏
- 显示关键配置参数

```
┌──────────────────────────────────────────────────┐
│ 📄 混沌仙途                    🧠 智能记忆 · 16,189字 │
│                                                  │
│ 解析配置：标准模式 · 深度求索V3 · 国产短剧规则      │
│ [修改配置]                                       │
└──────────────────────────────────────────────────┘
```

**优点**：

- 信息完整，一目了然
- 可快速修改配置
- 适合复杂配置场景

### 做法 3：重新解析确认弹窗（参考 Google Docs、飞书）

**参考**：Google Docs 重新发布确认、飞书文档重新生成

**实现**：

- 点击"重新解析"时弹出确认框
- 显示当前配置，允许调整

```
┌─────────────────────────────────────────┐
│           确认重新解析                   │
├─────────────────────────────────────────┤
│                                         │
│ 当前配置：                              │
│   🧠 智能记忆        [开关] 已启用      │
│   📊 解析模式        [下拉] 完整解析    │
│   🤖 AI 模型         [下拉] 深度求索V3  │
│   📝 字数            16,189 字          │
│                                         │
│   [取消]              [开始解析]        │
│                                         │
└─────────────────────────────────────────┘
```

**优点**：

- 明确告知用户当前配置
- 可一键调整，无需返回设置
- 防止误操作

### 做法 4：历史配置记录（参考 Photoshop、Premiere）

**参考**：Adobe 系列历史记录面板

**实现**：

- 记录每次解析的配置
- 显示解析历史对比

```
┌─────────────────────────────────────────┐
│ 解析历史                                │
├─────────────────────────────────────────┤
│                                         │
│ 🕐 刚刚                                 │
│   智能记忆：开启  模型：深度求索V3       │
│   结果：31个分镜  质量：92分            │
│                                         │
│ 🕑 2小时前                              │
│   智能记忆：关闭  模型：深度求索V3       │
│   结果：28个分镜  质量：85分            │
│                                         │
│ [使用此配置重新解析]                    │
│                                         │
└─────────────────────────────────────────┘
```

**优点**：

- 可追溯历史配置
- 方便对比不同配置效果
- 适合反复调优场景
