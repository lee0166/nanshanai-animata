# 文本清洗问题分析 V2

## 用户反馈

用户上传小说后，剧本管理页面的"原文"中仍然包含大量特殊字符（如 `///////////`），清洗功能没有按预期工作。

## 截图证据

```
阳光穿过初夏的薄雾洒在花园的///////////木椅上，
柳唯穿着亮黄色蓬蓬///裙从花园后门推门进来，

手里还拿着一///杯热咖啡，
微笑着说："李叔叔，今天终于蹲到你了"。
```

**问题**：`///` 这样的特殊字符没有被清洗掉。

## 我的错误分析

### 之前的错误结论

我错误地认为清洗功能已经正确实现，只是没有保留原始文本。实际上：

- ❌ 清洗功能代码存在
- ❌ 但清洗逻辑不完整
- ❌ 没有处理小说中的特殊分隔符

### 实际原因

`TextCleaner.clean` 方法只处理了：

- ✅ 换行符统一
- ✅ 控制字符去除
- ✅ 空格规范化
- ❌ **没有处理 `/`、`*`、`-` 等特殊字符**

## 正确的修复方案

### 方案1：增强清洗逻辑（推荐）

在 `TextCleaner.clean` 方法中添加对特殊字符的处理：

```typescript
static clean(text: string): string {
  // ... 现有清洗逻辑 ...

  // 8. 去除多余的特殊字符（如 ///、***、--- 等分隔符）
  cleaned = cleaned.replace(/\/{2,}/g, '');  // 去除连续的 /
  cleaned = cleaned.replace(/\*{2,}/g, '');  // 去除连续的 *
  cleaned = cleaned.replace(/-{2,}/g, '');   // 去除连续的 -
  cleaned = cleaned.replace(/={2,}/g, '');   // 去除连续的 =

  // 9. 去除小说中的特殊标记（如 [注]、【作者说】等）
  cleaned = cleaned.replace(/\[.*?\]/g, '');  // 去除方括号内容
  cleaned = cleaned.replace(/【.*?】/g, '');  // 去除【】内容

  // 10. 去除多余的标点符号
  cleaned = cleaned.replace(/，{2,}/g, '，');  // 连续的逗号
  cleaned = cleaned.replace(/。{2,}/g, '。');  // 连续的句号
  cleaned = cleaned.replace(/！{2,}/g, '！');  // 连续的感叹号
  cleaned = cleaned.replace(/？{2,}/g, '？');  // 连续的问号

  return cleaned;
}
```

### 方案2：添加配置选项

让用户选择清洗级别：

- 轻度清洗：只处理换行符和空格
- 中度清洗：+ 控制字符 + 多余标点
- 深度清洗：+ 所有特殊字符
