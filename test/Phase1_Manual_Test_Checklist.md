# Phase 1 手动测试标准对照表

## 测试环境要求

- 浏览器：Chrome/Edge/Firefox 最新版本
- 后端服务：已启动并正常运行
- 测试剧本：准备1-2个不同风格的剧本文件（建议：现代都市剧、古装剧）
