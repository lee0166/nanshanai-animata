# 场景提示词生成问题分析与优化方案

## 问题描述

用户反馈：在场景管理页面中，选择不同场景时，提示词与场景应该有的提示词出入太大。

### 截图分析

**场景1：书桌/卧室**

- 提示词："深秋深夜的出租屋内，暖黄台灯下，女孩在书桌前疲惫工作，一只琥珀色眼睛的小奶猫蜷缩在她手边..."
- 问题：提示词过于具体，包含了角色（女孩、小奶猫）和动作（工作、蜷缩），这不是场景提示词应该有的内容

**场景2：厨房**

- 提示词："一个温馨的出租屋厨房，暖黄灯光下，女孩正在灶台前煮面，一只小奶猫蹲在脚边..."
- 问题：同样包含了角色（女孩、小奶猫）和动作（煮面、蹲），这是分镜/关键帧级别的描述，不是场景描述

## 核心问题诊断

### 问题1：场景提示词与分镜提示词混淆

**场景提示词应该是什么？**

- 场景 = 环境 + 空间 + 氛围 + 光线
- 不包含：角色、动作、情节

**当前问题**：

- 场景提示词包含了角色和动作
- 这会导致：
  1. 场景图中有固定角色，无法复用
  2. 场景图过于具体，不符合场景概念
  3. 与分镜/关键帧提示词重复

### 问题2：场景提示词生成逻辑错误

**推测的当前逻辑**：

```
场景名称 → AI生成 → 包含角色和动作的"场景描述"
```

**正确的逻辑应该是**：

```
场景名称 → AI生成 → 纯环境描述（无角色、无动作）
```

## 场景提示词的正确特征

### 场景提示词应该包含：

1. **空间环境**
   - 房间类型（卧室、厨房、客厅）
   - 空间大小（宽敞、狭小）
   - 布局（开放式、隔断式）

2. **视觉元素**
   - 家具（书桌、床、灶台）
   - 装饰（窗帘、地毯、挂画）
   - 颜色（暖色调、冷色调）

3. **光线氛围**
   - 光源（自然光、台灯、吊灯）
   - 时间（白天、黄昏、深夜）
   - 氛围（温馨、冷清、神秘）

4. **风格特征**
   - 装修风格（现代、复古、简约）
   - 材质（木质、金属、布艺）

### 场景提示词不应该包含：

1. ❌ 角色（女孩、小猫、男人）
2. ❌ 动作（工作、煮面、蹲）
3. ❌ 情节（疲惫、温馨互动）
4. ❌ 具体事件

## 对比示例

### ❌ 错误的场景提示词（当前）

```
深秋深夜的出租屋内，暖黄台灯下，女孩在书桌前疲惫工作，
一只琥珀色眼睛的小奶猫蜷缩在她手边，将她的指尖圈在怀里，画面温馨治愈。
```

**问题**：包含了角色（女孩、小奶猫）和动作（工作、蜷缩）

### ✅ 正确的场景提示词（应该）

```
深秋深夜的出租屋卧室，暖黄色台灯照明，木质书桌上摆放着笔记本电脑和书籍，
米色窗帘半掩，床铺整洁，墙面挂着简约装饰画，整体氛围温馨安静，适合工作学习。
```

**特点**：

- 只有环境描述
- 无角色
- 无动作
- 可复用性强

## 优化方案

### 方案1：修改场景提示词生成Prompt

**当前Prompt（推测）**：

```
请根据场景名称"{场景名称}"生成一个详细的场景描述，用于AI绘图。
```

**优化后的Prompt**：

```
请根据场景名称"{场景名称}"生成一个场景环境描述，用于AI绘图生成场景背景图。

要求：
1. 只描述环境、空间、氛围、光线
2. 不要包含任何角色（人物、动物）
3. 不要包含任何动作或情节
4. 描述应该是静态的，可复用的背景
5. 包含：空间类型、家具陈设、光线来源、色彩氛围、材质风格

示例：
- 场景名称："现代客厅"
- 正确描述："宽敞明亮的现代风格客厅，落地窗外是城市夜景，灰色布艺沙发，大理石茶几，暖黄色落地灯，墙面挂着抽象画，整体色调温馨舒适"
- 错误描述："一个男人坐在沙发上看电视，旁边的小狗在玩耍"（包含了角色和动作，不正确）

请生成场景描述：
```

### 方案2：增加场景提示词审核机制

在生成场景提示词后，自动检查：

1. 是否包含角色名称（人名、动物名）
2. 是否包含动作动词（坐、站、走、看等）
3. 如果包含，提示用户修改或重新生成

### 方案3：提供场景提示词模板

根据常见场景类型，提供预设模板：

| 场景类型 | 提示词模板                                     |
| -------- | ---------------------------------------------- |
| 卧室     | 温馨卧室，{光线}，{家具}，{装饰}，{氛围}       |
| 厨房     | 现代厨房，{光线}，{厨具}，{色调}，{氛围}       |
| 客厅     | 宽敞客厅，{光线}，{家具}，{装饰}，{氛围}       |
| 办公室   | 简约办公室，{光线}，{办公设备}，{色调}，{氛围} |

优化后：

- 场景提示词只包含环境描述
- 场景图可复用于多个分镜
- 场景与角色分离，便于组合
- 生成质量提升，更符合影视制作流程
