## 功能概述

本项目的大模型配置功能是一个完整的、可扩展的AI模型管理系统，支持：

- **多种模型类型**：图像生成、视频生成、文本解析(LLM)
- **多个Provider**：火山引擎、Vidu、阿里云、OpenAI、魔搭社区等
- **灵活的配置管理**：从预配置模板到完全自定义
- **UI配置界面**：搜索、筛选、编辑、删除等功能
- **配置验证**：内置验证机制确保配置正确性
- **版本管理**：配置历史记录和回滚
- **导入导出**：配置备份和分享

## 数据结构定义

### 核心类型定义 (types.ts)

#### 1. ModelConfig - 模型配置

```typescript
interface ModelConfig {
  id: string;                          // 配置唯一ID
  name: string;                        // 显示名称
  provider: string;                    // 提供商ID (volcengine/vidu/aliyun等)
  modelId: string;                     // 实际API模型ID
  type: 'image' | 'video' | 'llm';   // 模型类型
  
  // 能力配置
  capabilities: ModelCapabilities;
  
  // 参数配置
  parameters: ModelParameter[];
  
  // 运行时配置
  templateId?: string;                 // 关联的模板ID
  apiKey?: string;                     // API密钥
  isDefault?: boolean;                 // 是否为默认模型
  apiUrl?: string;                     // 自定义API地址
  baseUrl?: string;                    // 基础URL
  enabled?: boolean;                   // 是否启用
  providerOptions?: any;               // Provider特定选项
  
  // 价格配置 (可选)
  costPer1KInput?: number;            // 输入价格(美元/1K tokens)
  costPer1KOutput?: number;           // 输出价格(美元/1K tokens)
}
```

#### 2. ModelCapabilities - 模型能力

```typescript
interface ModelCapabilities {
  // 通用能力
  supportsReferenceImage?: boolean;      // 是否支持参考图
  maxReferenceImages?: number;           // 最大参考图数量
  maxBatchSize?: number;                  // 最大批量生成数
  appendCountToPrompt?: boolean;          // 是否在提示词后追加数量
  requiresImageInput?: boolean;           // 是否必需图片输入
  
  // 图像生成能力
  supportedResolutions?: string[];        // 支持的分辨率
  defaultResolution?: string;             // 默认分辨率
  minPixels?: number;                      // 最小像素数
  maxPixels?: number;                      // 最大像素数
  minAspectRatio?: number;                 // 最小宽高比
  maxAspectRatio?: number;                 // 最大宽高比
  supportedAspectRatios?: string[];        // 支持的宽高比
  
  // 视频生成能力
  supportsStartFrame?: boolean;            // 是否支持首帧
  supportsEndFrame?: boolean;              // 是否支持尾帧
  supportsAudioGeneration?: boolean;       // 是否支持音频生成
  supportedGenerationTypes?: (
    'text_to_video' | 'first_last_frame' | 'multi_ref'
  )[];                                      // 支持的生成类型
  
  // LLM能力
  maxContextLength?: number;               // 最大上下文长度
  supportsStreaming?: boolean;             // 是否支持流式输出
  supportsFunctionCalling?: boolean;       // 是否支持函数调用
  supportsJsonMode?: boolean;              // 是否支持JSON模式
  supportsSystemPrompt?: boolean;          // 是否支持系统提示词
  
  // 能力标注
  strength?: string[];                     // 模型擅长的场景
  bestFor?: string[];                      // 最佳使用场景
}
```

#### 3. ModelParameter - 模型参数

```typescript
interface ModelParameter {
  name: string;                            // API参数名
  label: string;                           // UI显示标签
  type: 'string' | 'number' | 'boolean' | 'select';  // 参数类型
  
  // select类型专用
  options?: { label: string; value: any }[];
  
  // 通用属性
  defaultValue?: any;                      // 默认值
  min?: number;                            // 最小值 (number类型)
  max?: number;                            // 最大值 (number类型)
  step?: number;                           // 步长 (number类型)
  description?: string;                    // 参数说明
  required?: boolean;                      // 是否必填
  
  // 可见性控制
  visibilityCondition?: {
    generationType?: string[];             // 仅在这些生成类型可见
    hasStartImage?: boolean;               // 有首帧时可见
    hasEndImage?: boolean;                 // 有尾帧时可见
  };
  
  hiddenCondition?: {
    generationType?: string[];             // 在这些生成类型隐藏
    hasStartImage?: boolean;
    hasEndImage?: boolean;
  };
}
```

### 常用参数定义

- `guidanceScale`: 引导系数 (1-20, 默认3.5)

#### 视频模型通用参数 (COMMON_VOLC_VIDEO_PARAMS)
- `resolution`: 分辨率 (480p/720p/1080p)
- `seed`: 随机种子
- `watermark`: 水印开关
- `cameraFixed`: 相机固定

#### LLM模型通用参数 (COMMON_LLM_PARAMS)
- `temperature`: 温度 (0-2, 默认0.3)
- `maxTokens`: 最大Token数 (100-128000)

## 配置管理界面

### 界面位置
文件：`views/Settings.tsx`

### 主要功能区域

#### 1. 模型列表展示
- **表格视图**：显示所有已配置的模型
- **搜索功能**：按模型名称或Model ID搜索
- **类型筛选**：图像/视频/LLM多选
- **提供商筛选**：按提供商筛选
- **排序**：按名称/类型/最近添加

#### 2. 添加模型
支持两种方式：

**方式一：从预配置模板添加**
1. 选择模型类型（图像/视频/LLM）
2. 选择Provider
3. 选择基础模型（从DEFAULT_MODELS中过滤）
4. 输入自定义名称
5. 输入API Key
6. 保存

**方式二：完全自定义模型**
1. 输入模型名称
2. 选择Provider
3. 输入Model ID
4. 配置API URL（可选）
5. 配置能力参数
6. 输入API Key
7. 保存

#### 3. 编辑模型
- 修改模型名称
- 修改Model ID
- 修改API Key
- 修改API URL
- 修改参数默认值
- 配置价格（LLM）

#### 4. 删除模型
- 二次确认删除
- 从settings.models中移除

## 模型模板系统

### 模板定义 (config/modelTemplates.ts)

#### 模板结构

```typescript
interface ModelTemplate {
  id: string;                          // 模板唯一标识
  name: string;                        // 模板显示名称
  type: ModelType;                     // 模型类型
  extends?: string;                    // 继承的父模板ID
  
  capabilities: ModelCapabilities;    // 模型能力
  parameters: ModelParameter[];       // 模型参数
  providerOptions?: ProviderSpecificOptions;
  
  environment: Environment;            // 环境限制
  version?: string;                    // 模板版本
  
  deprecated?: boolean;                // 是否弃用
  deprecationMessage?: string;         // 弃用提示
  replacementTemplateId?: string;      // 替代模板ID
}
```

### 预置模板

#### 1. 基础模板

- **BASE_IMAGE_TEMPLATE**: 基础图像生成模板
- **BASE_VIDEO_TEMPLATE**: 基础视频生成模板

#### API使用示例

```typescript
import { modelConfigManager } from '../services/ai/core/ModelConfigManager';

// 1. 从模板创建

  'template-volcengine-image',

if (result.valid) {

// 2. 自定义创建

  apiKey: 'sk-xxx...',
  capabilities: {
    maxContextLength: 128000,
    supportsStreaming: true,
  },
});

// 3. 更新配置
modelConfigManager.updateConfig(configId, {
  name: '更新后的名称',
  apiKey: 'new-key...',
});

// 4. 查询配置
const config = modelConfigManager.getConfig(configId);
const allConfigs = modelConfigManager.getAllConfigs();
const enabledConfigs = modelConfigManager.getEnabledConfigs();

// 5. 版本管理
const versions = modelConfigManager.getConfigVersions(configId);

// 6. 导入导出
const exportData = modelConfigManager.exportConfigs();

```

## Provider系统

### Provider架构

每个Provider实现 `IAIProvider` 接口：

```typescript
interface IAIProvider {
  id: string;
  
  generateImage(
    prompt: string,
    config: ModelConfig,
    referenceImages?: string[],
    aspectRatio?: string,
    resolution?: string,
    count?: number,
    guidanceScale?: number,
    extraParams?: Record&lt;string, any&gt;
  ): Promise&lt;AIResult&gt;;
  
  generateVideo(
    prompt: string,
    config: ModelConfig,
    startImage?: string,
    endImage?: string,
    existingTaskId?: string,
    onTaskId?: (id: string) =&gt; void,
    extraParams?: Record&lt;string, any&gt;
  ): Promise&lt;AIResult&gt;;
  
  generateText?(
    prompt: string,
    config: ModelConfig,
    systemPrompt?: string,
    extraParams?: Record&lt;string, any&gt;
  ): Promise&lt;AIResult&gt;;
  
  validateConfig(config: ModelConfig): boolean;
}
```

### 现有Provider

#### 1. VolcengineProvider (火山引擎)
- 位置：`services/ai/providers/VolcengineProvider.ts`
- 支持：图像生成、视频生成
- 策略系统：Seedream4Strategy、Seedream3Strategy、DefaultStrategy

#### 2. ViduProvider
- 位置：`services/ai/providers/ViduProvider.ts`
- 支持：图像生成、视频生成

#### 3. AliyunVideoProvider (阿里云)
- 位置：`services/ai/providers/AliyunVideoProvider.ts`
- 支持：视频生成

#### 4. ModelscopeProvider (魔搭社区)
- 位置：`services/ai/providers/ModelscopeProvider.ts`
- 支持：图像生成（开发测试用）

#### 5. LLMProvider
- 位置：`services/ai/providers/LLMProvider.ts`
- 支持：文本生成
- 兼容：OpenAI协议的所有LLM

### 故障排查

#### 常见问题

**Q: 模型生成失败怎么办？**
A: 检查以下几点：
1. API Key是否正确
2. API URL是否可访问
3. 账户是否有足够额度
4. 模型ID是否正确

**Q: 如何查看模型能力？**
A: 在Settings页面查看模型列表，或者查看 `config/models.ts` 中的 `capabilities` 字段。

**Q: 可以添加多个同类型模型吗？**
A: 可以，添加时使用不同的名称即可。
